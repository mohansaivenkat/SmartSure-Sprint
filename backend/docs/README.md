# Documentation (API and Architecture)

This folder contains documentation related to the SmartSure Insurance Management System. Typically, this includes OpenAPI/Swagger specifications and architectural diagrams.

##  Contents

- **API Specifications**: Exported JSON/YAML files for the various microservices' APIs.
- **Visual Diagrams**: Architecture, sequence, and component diagrams if available.

##  Accessing Live API Documentation

You can access the live OpenAPI documentation (Swagger UI) for each service when it is running:

- **Auth Service**: `http://localhost:8081/swagger-ui.html`
- **Policy Service**: `http://localhost:8082/swagger-ui.html`
- **Claims Service**: `http://localhost:8083/swagger-ui.html`
- **Admin Service**: `http://localhost:8084/swagger-ui.html`
- **API Gateway**: `http://localhost:8080/swagger-ui.html`
